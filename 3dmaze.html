3D Maze Game

The 3D Maze Game you see above inserts the player into labyrinthine mazes and tasks them with finding all of the golden orbs hidden throughout the level so that they can escape... to the next level. The game features a minimalistic art style that I designed myself, similar to that of SUPERHOT. Everything within the game is clearly color coded, with a white environment, red decorations, golden collectibles and a blue exit. I would have liked to continue improving the game, but unfortunately I no longer have access to the source code.

While I no longer have access to the source code of the game, there is still some gameplay footage of the game that was saved online instead:

The 3D Maze Game was developed in Unreal Engine 4 and coded using Unreal's Blueprints system.
