Recent Projects

Pickleland Quest

Pickleland Quest is a 2D sidescroller platformer developed using Unity Engine and C#. In it, the player must traverse perilous levels to reach the end, all the while collecting coins, avoiding hazards and defeating enemies. The game is currently in development and should be finished soon.

Turn-Based RPG Battle System

In this customizable battle system, the player does battle with an AI-controlled enemy, both of whom have unique attacks and abilities at their disposal. Their stats can be altered to create different fights and scenarios each time. The system was developed using Unity Engine and C#.

Maze Game

This 3D Maze game was developed in Unreal Engine 4 using the Blueprints system. It features multiple levels that the player must thoroughly search to find golden orbs, which allow them to reach the next level.

  • More

Text-based Dice roll game

In this text-based game written in Python, two players take turns rolling dice to see who can get a higher score, with some twists thrown in. The game features an account system, and saves the winner of each game along with their score to a leaderboard.
